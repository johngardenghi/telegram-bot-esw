# Bot do Telegram para atribuição de orientadores de estágio

Este bot é destinado para atribuição de orientadores de estágio do curso de Engenharia de Software.

Os pacotes necessários para seu funcionamento estão em requirements.txt.

Para utilizar, é necessário configurar as seguintes variáveis de ambiente:

```
ESWBOT_GMAIL_USER: usuário do Gmail para envio de e-mail
ESWBOT_GMAIL_PASSWORD: senha do Gmail para envio de e-mail
ESWBOT_TOKEN: Token do bot no Telegram
ESWBOT_DB_HOST: endereço do servidor de banco de dados (MySQL)
ESWBOT_DB_USER: usuário do banco
ESWBOT_DB_PASS: senha do banco
ESWBOT_SIGAA_USER: usuário do SIGAA
ESWBOT_SIGAA_PASS: senha do SIGAA
```
